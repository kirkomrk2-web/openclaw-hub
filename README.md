# OpenClaw Hub 🦞

MoltBot Showcase Dashboard — операционен център за Молти и цялата Wallestars инфраструктура.

## 📊 Страници

- **Dashboard** — Статистики, activity feed, quick actions, Tailscale mesh визуализация
- **Calendar** — Планиране на събития и задачи  
- **Resources** — Управление на ресурси и мониторинг
- **Watchtower** — Системен мониторинг и алерти
- **Playground** — Интерактивен терминал и тестове

## 🛠 Stack

- React 19 + React Router 7
- Tailwind CSS с custom glassmorphism theme
- Framer Motion за анимации
- Shadcn/ui компоненти
- Lucide Icons
- Sonner notifications

## 🚀 Development

```bash
cd frontend
npm install
npm start
```

## 📦 Build & Deploy

```bash
cd frontend
npm run build
# Deploy build/ folder to Netlify
```
